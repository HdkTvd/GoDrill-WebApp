# GoDrill-WebApp

## A Project in GoLang.

### How to execute this project?

Clone this repo -
`git clone https://github.com/HdkTvd/GoDrill-WebApp.git`

Go to your project directory -
`cd ./GoDrill-WebApp`

Install Dependencies -
`go mod tidy`

In your project directory run command -
`go run main.go`
